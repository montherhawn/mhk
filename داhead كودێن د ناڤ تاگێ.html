شەرحکرنا كودێن د ناڤ تاگێ head دا

زمانێ html ژ دوو پەشێن گرنگ یێن ئەساسی پێک دهێن ئەو ژی head و body مە ل ڤێرێ شەرحکریا و دڤێ شەرحێ دا ئەمێ هەموو تاکێن د پەشێ ئێکێ دا ئەو ژی تاکێ head شەرحکەن.

1 HTML head tag
<head> </head>
مە بەری نهۆ تاگێ head شەرح کریا ،شەرح
2 HTML title tag
<title> </title>
مە بەری نهۆ تاگێ title شەرح کریا ،شەرح
3 HTML style tag
<style> </style>
4 HTML main tag
<main> </main>
ئەڤ تاگ گەلەك تاگ دناڤ دەینە یێن ئەساسیا مالپەرێ و نموونە ل سەر ڤێ مالپەرێ ؛
پرسیار/ ناڤەرۆکا ئەساسی یا مالپەری چییە؟
بەرسڤ/ ناڤەرۆکە ئەساسی ئەوا یا گرنگترین کۆدێن زمانێ html تێدا یە
پێزانینێن سەرەکی د ناڤ رووپەلێ دەینە و بێ وێ رووپەلە مە ج واتەیا( مەعنا ) نادە ؛ و دنیڤا رووپەلێ دە دناڤبەرا Header وFooter
5 HTML aside tag
<aside> </aside>
  • aside مەعنا وی ئاليەک یان رەخەك
  • ئەو پەشەکە ژ رەخێ مالپەری و بتریا جاران پێکدهێت ژ ریکلاما بەس با تو د بینیSidebar ل مالپەرێ مە دا پێکدهێت ژ لینکێت فێرکرنا زمانێ پرۆگرامینکێ.
  • ⇨ ئەڤە جهێSidebar
  • هەردیم په‌یوه‌ندیەك دناڤبەرا aside و مالپەرێ دا هەیە وەکی سایتێ مە.
6 HTML footer tag
<footer> </footer>
ئەو پەشێ دوماهیکێ یە ژ مالپەری و ل بنێ مالپەری یە پێکدهێت ژ:
  • لینکێت پەیجێن تۆرێن جڤاکی یێن مالپه‌رێ
  • لینکێت سایتا دناڤ مالپەرێ دا
و دڤێ حالەتێ دە مالپەرێ مە ب رستەکە جوان و سادە هاتیە جێکرن ♥

7 HTML article tag
<article> </article>
  • ئەڤ تاگ دهێنە دیار کرن وەکی تاگێن سەربخۆ یەعنی هەکە تو وێ ژ رووپەلێ ژێبی ژی چ تەسیر ل رووپەلێ نابیت
  • هەروەسا دناڤ دا گەلەك کۆد دهێنە بکارئینان
جوداهی دناڤبەرا تاگێ <p>و تاگێ <article> چییە ؟
تاگێ <p> پاراگرافه‌ک د گۆتاره‌کێ دايه و تاگێ<article>ئەو بخۆ گۆتارە بو نموونە 👇🏻
<article>
<h1> كورتكرن </h1>
<p> ئه‌م ڤێ گۆتارێ د 3 تاگان ده‌ کورت دکەن.... </p>
<article/>
8 HTML section tag
<section> </section>
section مەعنە وێ لق( قسم ، فرع ) و گەلەك جاران رووپەلێ داپەش دکە ل سەر چەند بەشە
جوداهی دناڤبەرا تاگێ <section>و تاگێ <article> چییە ؟
نموونە ل سەر ڤێ چەندێ دەمێ تە سایتەك هەبیت و ژ دوو قسمە پێکهاتبیت:
  • قسمێ یەکێ ل سەر نووچه‌یێن سپۆرێ را بيت ئەم <section> بکاردئینن
  • جارا یەکێ ئەم تاگێ <article> ب کاردئینن
  • جارا دووێ ئەم تاگێ <article> ب کاردئینن
  • قسمێ دووێ ل سەر نووچه‌یێن هه‌وایێ ( حالة الطقس ) را بيت ئەم <section> بکاردئینن
  • جارا یەکێ ئەم تاگێ <article> ب کاردئینن
  • جارا دووێ ئەم تاگێ <article> ب کاردئینن
9 HTML figure; figcaption; img tags
<figure> </figure>
<figcaption> </figcaption>
<img src="example.jpg">
دەمێ تە بڤێت تو وێنەکی د ناڤ سایتێ خۆ دە زێدا بکەی و رستەکێ ل بنێ وێنێ بنڤیسی رێکا درست بۆ بکارئینانێ ئەڤ هەرسێ تاگن👆🏽. بو نموونە تە وێنێ لالش هەبیت lalish.png و ل بنی دە بنڤیسی لالشە نورانی ,ئەڤ کۆدێن وێ نە 
 
            <figure>
                <img src="lalish.png">
                <figcaption> لالشا نورانى  </figcaption>
            </figure>
و ژبەر ڤێ چەندێ کۆدێ <figure>پیێك دهێت ژ :
  • بۆ وێنە ئەم کۆدێ <img> ب کاردئینن
  • بۆ شەرحکرنا وێنەی ئەم کۆدێ <figcaption> بکاردئینن
10 HTML time tag
<time> </time>
ئەڤ تاك نەی محتاجە شەرح بکەن😁
هەکە مە بڤێت ئەم دەمەکی تایبەت بنڤیسن دڤێت ئەم ڤی تاکی بکاربینن بو نموونە:
<time> 12:45 </time>
و attribute خۆ یێ تایبەت ئەو ژی detetime و بتريا جاران ب زمانێ JavaScript دهێتە بکارئینان و رێکە ب کارئینانا ڤی تاکی ئەڤەیە 👇🏻: 
 
                <p> 
                  <time datetime="1990-07-25 22:00"> يوم عيد ميلادى </time> 
                  ياتى فى نفس المعاد كل عام ههخه
