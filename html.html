ئەساسیاتێت زمانێ HTML

1 HTML چییــە ؟
دەستپێکێ زمانێ HTML نە زمانکی برمجیە بەلکو زمانەکێ هایپەر مارکئەپە (Hyper Markup) و ب بنیاتێ مالپەری دهێتە دانانێ کا چاوا دەمێ خانیەکێ چێکەی تۆ پێتڤی ب ئەساسێ خانیێ دەستپێکێ چێکەی هۆسا بۆ مالپەری ژی تۆ پێتڤی ب زمانێ HTML داکۆ کۆدێن دەستپیکێ بنڤیسی و بشێی کارێ خۆ تەمام بکەی .
HTML زمانەکێ سادە و ب ساناهی یە ، پێتڤی ب گەلەک دەمی نابیت بۆ فێربوونا وێ.
HTML بەیڤەکه هاتیە کورتکرن ژ رستە Hyper Text Markup Language.
2 مێژوویا HTML
زمانێ HTML ل سالا 1991 هاتبوو چێکرن ژلایێ ماموستایێ مەزن تیم بێرنەرس لیی(Tim Berners Lee) ڤە دەمێ ئێکەم مالپەر لسەر ئینتەرنتێ بەلاڤ کری ب رێکا بکارئینانا زمانێ HTML بوو، و ل ساڵە 1995 کۆمه‌کە پرۆگراممەرا ئه‌و زمان پێش ئێخست و کرنا HTML 2

3.1 Header

1- لۆگۆیێ مالپەرێ یان ژی ناڤێ مالپەرێ
2- Navigation Bar
3- چارکۆشەیا لێگەریانێ ژبۆ لێگەریانە هەر تشتەکی د ناڤ مالپەرێ دە.
د ئەڤێ حالەتێ دە مالپەرێ مە وەک Header پێک دهێت ژ :

3.2 Navigation Bar
ئەو پەشەکە ژ Header و هندك جاران Header تایبەت ب مالپەرێ ڤە پێکدهێت ژNavigation Bar ب تنێ. و ئەو پێکدهێت ژ لینکە تو دکاری ب رێکا وی ژ مالپەرەکێ بجی ئێکا دی و بتریا مالپەرێ سەرەکی پێکدهێتن ژ :
1- About us
2- contact us
3- Home

3.3 Main content
